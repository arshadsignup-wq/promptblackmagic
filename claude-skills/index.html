How to Install Claude Skills

1

Clone the Skill

Clone or download the skill repository from GitHub to your local machine.

git clone https://github.com/author/skill-name
2

Add to Your Project

Copy the skill's CLAUDE.md or SKILL.md file into your project's .claude/ directory, or place it in the project root.

cp skill-name/CLAUDE.md .claude/
3

Use in Claude Code

Claude Code automatically detects skills. Use slash commands like /skill-name or simply describe what you need.

claude> /plan my new feature
