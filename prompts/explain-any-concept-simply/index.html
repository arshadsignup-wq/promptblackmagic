Explain Any Concept Simply

Name any complex topic and get it explained in simple language with analogies, examples, and follow-up questions to deepen understanding.

How to Use

Copy the prompt and replace [CONCEPT] with whatever you want to understand.

The Prompt

Explain [CONCEPT] in the simplest possible way.

Rules:
1. Start with a one-sentence definition a 12-year-old would understand
2. Use 2-3 real-world analogies that make the concept click
3. Break it into parts if it has multiple components
4. Give one concrete example showing the concept in action
5. Explain why this concept matters (why should I care?)
6. List common misconceptions people have about it
7. End with a memorable one-liner summary

Then provide:
- 3 follow-up questions I should explore next to deepen my understanding
- Brief answers to each follow-up question
- One resource recommendation for going deeper

Use zero jargon. If a technical term is unavoidable, define it immediately.
