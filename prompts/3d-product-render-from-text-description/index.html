3D Product Render From Text Description

Describe any product and get a detailed prompt for generating photorealistic 3D renders with studio lighting, material specs, and camera angles.

By The Prompt Black Magic Team

How to Use

Paste into Midjourney, DALL-E 3, or any image generation AI. Adjust the bracketed placeholders to match your product.

The Prompt

You are a senior 3D visualization artist who creates photorealistic product renders for global consumer brands.

Product: [PRODUCT NAME AND TYPE]
Material/finish: [MATTE/GLOSSY/METALLIC/WOOD/FABRIC/OTHER]
Color palette: [PRIMARY AND ACCENT COLORS]
Target audience: [WHO THIS PRODUCT IS FOR]

Generate a complete set of image generation prompts:

**1. Hero Shot Prompt**
- Write a detailed prompt for the primary marketing image
- Include: camera angle (3/4 view, eye-level, top-down), lighting setup (three-point, rim light, soft diffused), background (gradient, lifestyle setting, pure white), depth of field, render style
- Specify resolution and aspect ratio

**2. Detail Close-Up Prompts (3 variations)**
- Macro shots highlighting texture, craftsmanship, and unique features
- Include material rendering instructions (subsurface scattering for translucent materials, anisotropic reflections for brushed metal)

**3. Lifestyle Context Prompt**
- Product shown in its natural use environment
- Include props, setting details, time of day, mood lighting

**4. Exploded View / Technical Prompt**
- Deconstructed view showing internal components or layers
- Clean technical illustration style with labeled callouts

**5. Color Variant Prompts**
- Generate the same hero shot in 3 different colorways
- Maintain consistent lighting and angle across all variants

**6. Social Media Crop Prompts**
- Square (1:1) for Instagram feed
- Vertical (9:16) for Stories/Reels
- Wide (16:9) for website hero banner

For each prompt, write it as a single copy-paste-ready prompt optimized for [MIDJOURNEY/DALL-E/STABLE DIFFUSION]. Include negative prompt suggestions where relevant.

Pro Tips for Image Generation

  • Reference specific art styles, lighting setups, and color palettes rather than vague descriptors like "beautiful" or "professional" for more consistent results.
  • Specify the aspect ratio and intended use (Instagram post, website hero, print) so the composition works for your platform.
  • Describe the mood with 5-8 specific adjectives rather than generic terms - "warm golden hour glow with soft bokeh" beats "nice lighting" every time.

When to Use This Prompt

  • You are designing marketing materials and need eye-catching visuals that match a specific brand aesthetic.
  • You need to visualize a concept, mockup, or prototype before investing in professional production.
  • You are exploring different visual styles for a rebrand and want to rapidly test multiple creative directions.

Expected Results

  • Platform-specific aspect ratios and composition guidelines for each intended use case.
  • Color palette recommendations with hex codes that complement the subject and intended brand aesthetic.
  • Detailed, copy-paste ready prompts for AI image generators with camera settings, lighting specs, and style keywords.

How to Customize This Prompt

  • Modify the camera and lens specs to match the look you want - wider for environmental, tighter for product detail.
  • Replace the product description with your actual product name, dimensions, materials, and key visual features.
  • Swap the style preference for your brand aesthetic - minimal, luxury, playful, bold, or organic.
