AI-Powered Content Repurposing Engine

Give it one piece of content (blog, podcast, video) and get a system for turning it into 15+ pieces across every platform.

By The Prompt Black Magic Team

How to Use

Paste into any LLM with your original content or a link to it. Get back a complete content multiplication system.

The Prompt

You are a content repurposing strategist who helps creators turn one piece of content into a week's worth of multi-platform content.

Original content type: [BLOG POST/PODCAST EPISODE/YOUTUBE VIDEO/WEBINAR/NEWSLETTER]
Original content topic: [TOPIC OR PASTE THE CONTENT]
Your platforms: [LIST ALL PLATFORMS YOU'RE ACTIVE ON]
Brand voice: [PROFESSIONAL/CASUAL/WITTY/AUTHORITATIVE]

Create a complete repurposing system:

**1. Content Extraction**
- Identify the 5-7 key ideas from the original content
- Pull out quotable one-liners (tweet-worthy)
- Identify data points and statistics
- Extract stories and examples
- Find contrarian or surprising takes

**2. Platform-Specific Transformations**

From one piece of content, create:

- **Twitter/X Thread** - 8-10 tweet thread with hook and CTA
- **LinkedIn Post** - Long-form thought leadership post
- **LinkedIn Carousel** - 10-slide carousel with text for each slide
- **Instagram Carousel** - 7-10 slides with design direction
- **Instagram Reel/TikTok Script** - 30-60 second video script
- **Email Newsletter** - Subject line + body with personal angle
- **Blog Post** (if original wasn't a blog) - 800-word SEO article
- **Podcast Talking Points** (if original wasn't audio) - 10-minute monologue outline
- **YouTube Short Script** - Under 60 seconds, vertical format
- **Reddit Post** - Platform-appropriate version (no self-promotion feel)
- **Quora Answer** - Answer a related question using this content
- **Pinterest Pin** - Pin title, description, and image direction
- **Facebook Post** - Community-style, discussion-generating version
- **Slide Deck** - 5-slide summary for presentations
- **Infographic** - Visual data representation direction

**3. Publishing Schedule**
- Day-by-day distribution calendar (which platform, which piece, what time)
- Staggering strategy (don't post everything at once)
- Cross-linking between pieces

**4. Repurposing SOP Template**
- Repeatable process for any future content
- Time estimate per piece
- Tools to use for each format
- Quality checklist before posting

Write ALL the actual content, not just descriptions of what to write.

Pro Tips for Agentic AI

  • Include success criteria so the agent knows when to stop iterating. Without clear goals, agents can loop indefinitely.
  • Request the agent to log its reasoning at each step. Transparent decision-making makes debugging and optimization much easier.
  • Break complex agent tasks into clear, sequential sub-tasks in your prompt - AI agents perform best with well-defined step-by-step workflows.

When to Use This Prompt

  • You need to orchestrate multiple AI models in sequence, where one model validates and refines another model's output.
  • You are designing an AI agent workflow that needs to autonomously research, analyze, and summarize information from multiple sources.
  • Your current AI automation breaks at decision points and you need branching logic that handles edge cases gracefully.

Expected Results

  • Fallback and escalation protocols that define exactly what happens when an agent step fails or times out.
  • Step-by-step agent workflow blueprints with decision points, error handling, and success criteria.
  • Tool and API integration plans that define which resources the agent accesses at each workflow stage.

How to Customize This Prompt

  • Add your specific tools and APIs (Zapier, Make, LangChain) so the agent blueprint uses your real tech stack.
  • Adjust the complexity level based on whether you need a simple linear agent or a multi-branch decision tree.
  • Include your error scenarios so the agent has explicit fallback instructions for situations unique to your workflow.
