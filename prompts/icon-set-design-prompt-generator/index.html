Image Generation

Icon Set Design Prompt Generator

Define your app or website and get AI prompts for a complete, consistent icon set with multiple sizes, states, and style variations.

By The Prompt Black Magic Team

How to Use

Use with Midjourney or DALL-E 3 for concept generation. Recreate final icons in Figma or Illustrator for production use.

The Prompt

You are a UI icon designer who has created icon systems for apps with 100M+ downloads including productivity tools, fintech apps, and social platforms.

Product: [APP/WEBSITE/OPERATING SYSTEM]
Icon style: [LINE/FILLED/DUOTONE/3D/GLYPH/OUTLINED FILLED]
Icon count needed: [12/24/48/100+]
Primary use: [NAVIGATION BAR/SETTINGS/FEATURES/ONBOARDING/MARKETING]
Design system: [MATERIAL/IOS/CUSTOM]

Generate a comprehensive icon set prompt system:

**1. Foundation Icons (12 essentials)**
- Home, Search, Settings, Profile, Notifications
- Add/Create, Edit, Delete, Share, Download
- Menu/Hamburger, Back/Close
- Consistent stroke width, corner radius, and optical sizing
- 24x24px grid with 2px padding

**2. Category/Feature Icons (12 custom)**
- Icons specific to your product's unique features
- Each tells the story of its function
- Recognizable without labels
- Unique enough to be ownable by your brand
- Works at 16px and 48px sizes

**3. State Variations**
- Default (inactive) state
- Active/selected state (filled vs outlined)
- Disabled state (reduced opacity)
- Loading state (animated direction)
- Notification badge overlay

**4. Size Adaptations**
- 16px: simplified, fewer details
- 24px: standard navigation use
- 32px: feature highlight, cards
- 48px: onboarding, empty states
- 64px+: marketing, hero sections

**5. Dark Mode Variants**
- Adjusted stroke weights for light-on-dark
- Color adaptations for dark backgrounds
- Maintaining visual weight parity with light mode

**6. Animated Icon Concepts (6 key icons)**
- Micro-interaction direction for key icons
- State transition animations (tap, toggle, loading)
- Lottie-compatible design direction
- Timing and easing descriptions

For each prompt: specify exact pixel grid, stroke width (1.5px or 2px), corner radius (2px), cap style (round), and join style (round). Include style consistency rules that apply across ALL icons in the set.

Pro Tips for Image Generation

  • Describe the mood with 5-8 specific adjectives rather than generic terms - "warm golden hour glow with soft bokeh" beats "nice lighting" every time.
  • Always specify the exact camera, lens, and aperture in your prompt - AI image generators produce dramatically better results with technical photography terms.
  • Include negative prompts to exclude unwanted elements like blurry backgrounds, extra fingers, or distorted text in your generated images.

When to Use This Prompt

  • You are building a social media presence and need consistent, on-brand visual content across multiple platforms.
  • You want to create unique art or illustrations for a project without hiring a graphic designer.
  • You are designing marketing materials and need eye-catching visuals that match a specific brand aesthetic.

Expected Results

  • Detailed, copy-paste ready prompts for AI image generators with camera settings, lighting specs, and style keywords.
  • Multiple variations of the same concept covering different styles, angles, and moods for A/B testing.
  • Complete negative prompt suggestions to avoid common AI image artifacts and unwanted elements.

How to Customize This Prompt

  • Adjust the lighting and mood descriptors to match your brand photography guidelines.
  • Add your brand colors as hex codes so the AI can incorporate them into backgrounds and styling.
  • Modify the camera and lens specs to match the look you want - wider for environmental, tighter for product detail.
