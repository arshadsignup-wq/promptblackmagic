Pixel Art Game Asset Prompt Pack

Define a game genre and get complete AI prompts for pixel art characters, environments, items, and UI elements in a consistent style.

By The Prompt Black Magic Team

How to Use

Use with Midjourney or Stable Diffusion pixel art models. Customize the game genre and style in brackets.

The Prompt

You are a pixel art game developer who has created art for indie hits on Steam and retro-style mobile games.

Game genre: [PLATFORMER/RPG/ROGUELIKE/STRATEGY/ADVENTURE]
Art resolution: [16x16/32x32/64x64 TILES]
Color palette limit: [8/16/32 COLORS OR SPECIFIC PALETTE NAME]
Theme: [MEDIEVAL FANTASY/SCI-FI/POST-APOCALYPTIC/CUTE/HORROR]

Generate a complete pixel art prompt set:

**1. Player Character Sprite Sheet**
- Idle animation (4 frames)
- Walk cycle (6-8 frames)
- Jump / attack / special action
- Front, side, and back views
- Consistent pixel scale with clear silhouette

**2. Enemy Character Set (4 types)**
- Common enemy (small, simple animation)
- Ranged enemy (projectile attack)
- Heavy/tank enemy (larger, slower)
- Boss character (detailed, multi-phase)
- Each with idle and attack frames

**3. Environment Tileset**
- Ground tiles (3 variations + edges)
- Platform tiles (left cap, middle, right cap)
- Background layers (parallax-ready, 3 depth layers)
- Interactive objects (doors, chests, levers, signs)
- Decorative props (trees, rocks, crystals, furniture)

**4. Item & Pickup Icons**
- Health/mana items
- Weapons (sword, bow, staff, gun)
- Armor pieces
- Key items / quest objects
- Currency / collectibles
- Consistent icon frame and size

**5. UI Elements**
- Health bar / mana bar design
- Dialog box frame
- Inventory grid
- Menu buttons (normal, hover, pressed states)
- Font style recommendation

**6. Effects & Particles**
- Hit effect / damage flash
- Explosion / impact
- Magic spell effects
- Dust cloud / landing particles
- Collectible sparkle

For each prompt: specify exact pixel dimensions, color count, transparency requirements, and animation frame count. Write prompts that produce consistent style across all assets.

Pro Tips for Image Generation

  • Reference specific art styles, lighting setups, and color palettes rather than vague descriptors like "beautiful" or "professional" for more consistent results.
  • Specify the aspect ratio and intended use (Instagram post, website hero, print) so the composition works for your platform.
  • Describe the mood with 5-8 specific adjectives rather than generic terms - "warm golden hour glow with soft bokeh" beats "nice lighting" every time.

When to Use This Prompt

  • You want to create unique art or illustrations for a project without hiring a graphic designer.
  • You are designing marketing materials and need eye-catching visuals that match a specific brand aesthetic.
  • You need to visualize a concept, mockup, or prototype before investing in professional production.

Expected Results

  • Platform-specific aspect ratios and composition guidelines for each intended use case.
  • Color palette recommendations with hex codes that complement the subject and intended brand aesthetic.
  • Detailed, copy-paste ready prompts for AI image generators with camera settings, lighting specs, and style keywords.

How to Customize This Prompt

  • Add your brand colors as hex codes so the AI can incorporate them into backgrounds and styling.
  • Modify the camera and lens specs to match the look you want - wider for environmental, tighter for product detail.
  • Replace the product description with your actual product name, dimensions, materials, and key visual features.
