Conversion Rate Optimization Audit

Describe your store and get a complete CRO audit framework - funnel analysis, heatmap insights, A/B test priorities, and quick wins to implement today.

By The Prompt Black Magic Team

How to Use

Paste into any LLM with your store URL or description. Use the output to systematically improve your conversion rate.

The Prompt

You are a CRO specialist who has helped e-commerce brands increase conversion rates from 1% to 4%+ through systematic testing and optimization.

Store URL or description: [YOUR STORE]
Current conversion rate: [PERCENTAGE]
Monthly traffic: [VISITOR COUNT]
Primary traffic source: [PAID ADS/ORGANIC/SOCIAL/EMAIL/DIRECT]
Biggest suspected leak: [WHERE YOU THINK PEOPLE DROP OFF]

Conduct a complete CRO audit:

**1. Funnel Analysis Framework**
- Homepage to collection page conversion
- Collection to product page conversion
- Product page to add-to-cart rate
- Cart to checkout initiated
- Checkout to purchase completion
- For each stage: benchmark rate, what to measure, where data lives
- Drop-off diagnosis: what typically causes abandonment at each stage

**2. Homepage Audit**
- Hero section: is the value proposition clear in 5 seconds?
- Navigation: can users find what they want in 2 clicks?
- Social proof: are trust signals above the fold?
- CTA clarity: is the primary action obvious?
- Mobile experience: same impact on small screens?
- Page speed: does it load in under 3 seconds?

**3. Product Page Audit**
- Image quality and variety (zoom, lifestyle, detail)
- Description clarity (benefits first, features second)
- Price presentation (value justification present?)
- Add-to-cart button (visible without scrolling, color contrast)
- Trust elements (reviews, guarantees, shipping info)
- Urgency elements (stock levels, time-limited offers)
- Cross-sell / upsell effectiveness

**4. Cart and Checkout Audit**
- Cart abandonment rate analysis
- Surprise costs (shipping, tax revealed too late)
- Guest checkout availability
- Payment options variety
- Form field count and necessity
- Error handling quality
- Mobile checkout experience
- Security signals

**5. Quick Wins (Implement This Week)**
- 10 low-effort, high-impact changes
- No code changes required where possible
- Copy improvements
- Layout adjustments
- Trust signal additions
- Expected impact estimate for each

**6. A/B Test Roadmap**
- Priority 1 tests (highest potential impact)
- Priority 2 tests (moderate impact, validates hypotheses)
- Priority 3 tests (refinement and optimization)
- For each test: hypothesis, variation, success metric, minimum sample size
- Testing calendar (run one test at a time for clean data)

**7. Analytics Setup Audit**
- Required events and tracking
- Enhanced e-commerce tracking verification
- Funnel visualization setup
- Cohort analysis configuration
- Attribution model recommendation

**8. Ongoing Optimization Process**
- Weekly metrics review checklist
- Monthly deep-dive analysis
- Quarterly strategic review
- Continuous feedback collection (exit surveys, chat logs, reviews)

Provide specific, actionable recommendations, not general advice. Each suggestion should include what to change, why, and expected impact.

Pro Tips for E-Commerce

  • Ask for A/B test ideas alongside each recommendation so you can validate changes before committing.
  • Include your shipping model and return policy details since they significantly impact conversion and copy strategy.
  • Request mobile-specific optimizations separately - mobile e-commerce has different UX requirements than desktop.

When to Use This Prompt

  • Your cart abandonment rate is above 70% and you need a multi-channel recovery system to recapture lost sales.
  • You are expanding to new sales channels (Amazon, social commerce) and need platform-specific product copy.
  • You want to increase average order value and need cross-sell and upsell strategies for your product pages.

Expected Results

  • Conversion-focused page layouts with persuasion elements, trust signals, and CTA positioning.
  • A/B test recommendations with specific variables to test and success metrics to track.
  • Platform-specific product copy optimized for Shopify, Amazon, and social commerce channels.

How to Customize This Prompt

  • Modify the pricing and discount strategy to match your margins and competitive positioning.
  • Add your current traffic and conversion data so recommendations prioritize the highest-impact changes.
  • Replace the product details with your actual product name, specs, price, and unique selling points.
