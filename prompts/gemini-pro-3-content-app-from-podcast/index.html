Gemini Pro 3 Content App From Podcast

Upload a podcast to Gemini and extract blog posts, social captions, newsletter content, quote graphics, and video clip ideas from it.

How to Use

1. In Google Gemini, enable Thinking mode and Canvas
2. Find an educational video or podcast and download the audio (you can use online tools to rip audio from YouTube videos).
3. Attach the audio clip (100mbs or under) to the chat, and copy and paste the prompt below.
‍

You can of course change any part of the prompt to help you create different types of content.

‍

If the created app shows any errors, it should enable you to fix them with the 'Fix Now' button that appears.

The Prompt

Create an interactive experience on canvas that covers the main topics discussed in this attached podcast in summary cards.

For each main topic card, create buttons that will trigger the following if clicked;

1. A high quality, educational LinkedIn post.

2. An engaging post for X.

3. A 600 word blog article

4. An educational Instagram post

5. A script for a 1 minute short form video

We’ll go through one card at a time. Do not stop the experience if a button is clicked, just perform the task inside the chat.
